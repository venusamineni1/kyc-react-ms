react-pdf / PDF.js

@react-pdf-viewer/core
Higher-level wrapper with built-in toolbar, zoom, rotate, thumbnails & text search. Analyst-ready document review experience.

react-image-crop
react-image-crop
Interactive crop/zoom on image uploads. Analyst can select the MRZ zone, face zone or signature zone before OCR to improve accuracy.
Placed in: Upload Modal → Step 2 (Image Zone)
🖊️
Annotorious
@annotorious/react
Polygon & rectangle annotations on document images. Analysts mark suspected tamper zones, photo-swaps or inconsistent fonts. Stored as W3C JSON.
Placed in: CaseDetails → Document Viewer (Fraud Review)
👁️
MediaPipe Tasks
@mediapipe/tasks-vision
468-point face landmark detection at 60fps via WASM. Blink + head-pose liveness. Runs entirely in browser — no video leaves device.
Placed in: Onboarding / Case Create → Selfie Capture

OpenSeadragon
openseadragon
Deep-zoom on 600 DPI passport scans. Smooth pan & zoom on ultra-high-res images without loading the full image at once.
Placed in: CaseDetails → Viewer (High-res ID scans)

OCR Result Overlay
tess4j (backend)
OCR runs on the Java backend (Tess4J). Extracted fields surface as an overlay panel next to the document viewer showing confidence scores.
Placed in: CaseDetails → OCR Data Panel (sidebar)

ZXing · mrz-parser · Tika
Backend signals (MRZ check digit pass/fail, EXIF tampering, barcode vs OCR mismatch) surfaced as colour-coded badges in the viewer sidebar.

Why Crop Zones?
📊 MRZ Zone — Use PSM 6 (uniform block), whitelist alphanumeric only. 99%+ accuracy.
👤 Face Zone — Send just the photo region to DeepFace for faster comparison.
✍️ Signature Zone — Extract signature contour for cross-doc comparison.
📮 Address Zone — Send address block to OpenNLP NER.
Backend OCR Pipeline
1. Apache Tika — detect MIME, extract EXIF
2. PDFBox — extract embedded text (digital PDFs)
3. OpenCV — deskew, binarize, upscale to 300 DPI
4. Tess4J (Tesseract 5) — OCR at PSM mode
5. MRZ parser — parse + validate check digits
6. ZXing — detect & decode PDF417 barcodes
EXIF Fraud Signals (Tika)
• GPS coordinates vs declared address
• Modified date newer than creation date
• Editing software tag (Photoshop, GIMP)
• Device model cross-reference
